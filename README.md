# KobitoKey_QWERTY

## Keymap

Combos: `Q+W` = ESC, `A+S` = 英数(LANG2), `D+F` = かな(LANG1), `J+K` = TAB

### Layer 0: DEFAULT (QWERTY)

```
┌──────┬──────┬──────┬──────────┬──────┐  ┌──────┬──────────┬──────┬──────┬──────────┐
│  Q   │  W   │  E   │    R     │  T   │  │  Y   │    U     │  I   │  O   │    P     │
├──────┼──────┼──────┼──────────┼──────┤  ├──────┼──────────┼──────┼──────┼──────────┤
│  A   │  S   │  D   │    F     │  G   │  │  H   │    J     │  K   │  L   │    ;     │
├──────┼──────┼──────┼──────────┼──────┤  ├──────┼──────────┼──────┼──────┼──────────┤
│  Z   │  X   │  C   │    V     │  B   │  │  N   │    M     │  ,   │  .   │    '     │
├──────┼──────┼──────┼──────────┼──────┤  ├──────┼──────────┼──────┼──────┼──────────┤
│LSHFT │ LALT │ LCMD │SPACE/L1  │LCTRL │  │ BSPC │ENTER/L2  │RSHFT │RSHFT │ ALT+/   │
└──────┴──────┴──────┴──────────┴──────┘  └──────┴──────────┴──────┴──────┴──────────┘
```
