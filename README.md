## Keymap

![Keymap](images/keymap.svg)

## 設計方針

- プログラミング（Vim + Mac）向けに最適化
- Layer 2でShift併用を活用し、20キーで40種の記号をカバー（Layer 4不要）
- BT操作はLayer 3（MOUSE）に隔離して誤爆防止
- コンボでESC/TAB/英数/かなをLayer 0から直接入力可能

## 評価

### 良い点

- **Layer 0**: 素直なQWERTY。コンボでESC/TAB/英数/かなをカバーし追加キー不要
- **Layer 1**: 矢印がVimのhjkl位置で直感的。Home/End/PgUp/PgDnが矢印直下で覚えやすい
- **Layer 2**: Shift併用で全括弧`() {} [] <>`と全記号をカバー。シンプルかつ十分
- **Layer 3**: BT操作を普段触らないレイヤーに隔離して安全。オーディオとマウスの同居も自然

### 改善候補

| 項目 | 内容 | 優先度 |
|---|---|---|
| RSHFTが2つ | 片方を別の用途に変えれば1キー有効活用できる | 低 |
| Layer 2でShift+数字が3キー同時 | ENTER長押し+SHIFT+数字で`!@#`等。慣れは必要 | 中 |
| Layer 1の左手2-3段目が空き | 10キー分空いている。シンプルさとのトレードオフ | 低 |
